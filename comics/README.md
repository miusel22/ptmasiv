## 
Esta aplicación fue realizada con:
- ReactJs
- Javascript
- NodeJS
- Ant design
- Redux
- Less
### ¿Cómo se configura?

## FRONTEND  
1. clonar el repositorio
2. git flow init
3. Rama develop
4. npm i
5. npm start
## BACKEND 

1. Navegar hasta la carpeta src/backend
2. Correr el comando node server.js
